Adding colour

Fictoan comes with 223 different variables for adding colour to your elements. There are five propsbgColour, textColour, borderColour, strokeColour and fillColourthat accept any value from the list of colours below. Use them like so



<Element as="section" bgColour="indigo-40" />
    
<Heading textColour="crimson">Hello</Heading>
    
<Text textColor="slate-80">Hello</Text>
    
<Card borderColor="indigo-60" />
    
<SVG strokeColour="teal-90" />
    
<SVG fillColor="orange-20" />

The last two propsstrokeColour and fillColourwork only for SVG elements.

And oh, you can use both spellings of colourbgColour or bgColor as props. This applies for all five props above.
