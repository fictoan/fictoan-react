The idea

Fictoan focuses on code readability, and is primed for ease and speed of iteration.
Props, attributes and values are all plain English.
We want designers also to be able to contribute in codethey can focus on layout, theming and pixel-perfection while engineers can focus on the integrationthis saves time and headaches.
