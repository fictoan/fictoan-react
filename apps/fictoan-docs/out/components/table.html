Default

The Table element is used to separate sections of content on your page. Its a self-closing element, and doesnt accept any children.

<Table>
    <thead>
        <tr>
            <td>Header 1</td>
            <td>Header 2</td>
            <td>Header 3</td>
        </tr>
    </thead>


    <tbody>
        <tr>
            <td>Cell 1</td>
            <td>Cell 2</td>
            <td>Cell 3</td>
        </tr>
        <tr>
            <td>Cell 4</td>
            <td>Cell 5</td>
            <td>Cell 6</td>
        </tr>
    </tbody>
</Table>

The above code will display the default Table, which looks like this

Header 1Header 2Header 3
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6

The default table looks nothing like an actual tablewe need to add props to dress it up.

Props

Cell borders

You can add borders to the cells by adding the bordersFor prop. You can have rows, columns or both.

<Table bordersFor="both" />
Header 1Header 2Header 3
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6
Cell 7Cell 8Cell 9
Cell 10Cell 11Cell 12
Cell padding

Padding adds some spacing to the cells, and accepts the tiny to huge spacing values.

<Table bordersFor="both" padding="tiny" />
Header 1Header 2Header 3
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6
Cell 7Cell 8Cell 9
Cell 10Cell 11Cell 12
Striped rows

The isStriped prop adds a different background colour to alternate rows, aiding readability. It also adds a coloured header row. Both these values can be set in the theme.

<Table bordersFor="both" padding="tiny" isStriped />
Header 1Header 2Header 3
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6
Cell 7Cell 8Cell 9
Cell 10Cell 11Cell 12
Change row BG on hover

The isHoverable prop adds a different background colour to a row on hover. You can change the colour again in the theme.

<Table bordersFor="both" padding="tiny" isStriped />
Header 1Header 2Header 3
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6
Cell 7Cell 8Cell 9
Cell 10Cell 11Cell 12
Full-width

You can add the isFullWidth prop to make the table take up the width of its parent. This is a common prop and now unique to the Table element.

<Table bordersFor="both" padding="tiny" isStriped isHoverable isFullWidth />
Header 1Header 2Header 3
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6
Cell 7Cell 8Cell 9
Cell 10Cell 11Cell 12
Text alignment

Theres a alignText prop to align the text to the left, right, and centre/center.

<Table
    bordersFor="both" padding="tiny"
    isStriped isHoverable isFullWidth
    alignText="centre"
/>
Header 1Header 2Header 3
Cell 1Cell 2Cell 3
Cell 4Cell 5Cell 6
Cell 7Cell 8Cell 9
Cell 10Cell 11Cell 12

Theming

table : {
    bg      : defaultColours.white,
    text    : customColours.shade,
    border  : defaultColours.slate40,
    striped : {
        header : {
            bg : defaultColours.blue40,
        },
        cell : {
            bg : defaultColours.slate20,
        }
    },
    onHover : {
        bg   : defaultColours.amber20,
        text : customColours.shade
    },
}
